# Ultimate Portfolio Landing Page 🚀

A high-converting, modern, and fully responsive landing page designed for digital products and agencies. Built from scratch to showcase clean code architecture, modern UI/UX principles, and smooth browser animations.

## 🌐 Live Demo
[Click here to view the live site](https://nazarhlbv.github.io/example-landing/)

## ✨ Key Features

This landing page is packed with modern web development techniques and interactive elements:

* **Smooth Scroll Reveal Animations:** Utilizes the `IntersectionObserver` API to elegantly fade and slide in elements (like feature cards and testimonials) as the user scrolls down the page. Optimized with thresholds to ensure a natural feel.
* **Interactive Mobile Navigation:** A custom-built, responsive "hamburger" menu for mobile devices. It features smooth toggle animations and automatically closes when a user clicks a navigation link.
* **Dynamic FAQ Accordion:** An expandable/collapsible frequently asked questions section built with Vanilla JavaScript and CSS `max-height` transitions for a seamless user experience.
* **Modern UI/UX Design:** Includes a sticky header with a drop-shadow, CSS Grid layouts for perfect alignment, and scalable vector graphics (SVG) for crisp, high-quality icons.
* **Simulated Contact Form:** A fully styled contact form with basic validation and JavaScript event handling (`preventDefault`) that provides real-time feedback to the user upon submission.

## 🛠️ Tech Stack

* **HTML5** - Semantic and accessible markup.
* **CSS3** - Custom variables (`:root`), Flexbox, CSS Grid, Media Queries (Mobile-first approach), and smooth transitions.
* **Vanilla JavaScript (ES6+)** - DOM manipulation, Event Listeners, and Intersection Observer API (Zero dependencies).

## 🚀 How to Run Locally

No build tools or package managers are required. To view this project:

1. Clone this repository or download the ZIP file.
2. Open the extracted folder.
3. Double-click the `index.html` file to open it in your browser.
4. *Tip: Resize your browser window to test the responsive mobile layout and the hamburger menu!*
